# GoSTL

GoSTL is a algorithm and data structure library for go, designed to provide functions similar to C++ STL.

## Functions
- container
    - array
    - vector
    - deque
    - queue
    - priority_queue
    - stack
    - map
    - multimap
    - set 
    - multiset
    - rbtree
- algorithm
    - sort
    - stable_sort
    - lower_bound
    - upper_bound
