# GoSTL

GoSTL is a data structure and algorithm library for go,designed to provide functions similar to C++ STL,but more powerful.

## Function list
- containers
    - slice
    - array
    - vector
    - list
    - deque
    - queue
    - priority_queue
    - stack
    - rbtree
    - map/multimap
    - set/multiset
    - bitmap
    - bloom_filter
    - hamt(hash_array_map_tree)
    - ketama
    - skiplist
- algorithm
    - sort(quick_sort)
    - stable_sort(merge_sort)
    - binary_search
    - lower_bound
    - upper_bound
    - next_permutation
