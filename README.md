# GoSTL

GoSTL is a algorithm and data structure library for go, designed to provide functions similar to C++ STL.

## Functions
- container
    - array
    - vector
    - list
    - deque
    - queue
    - priority_queue
    - stack
    - rbtree
    - map
    - multimap
    - set 
    - multiset
- algorithm
    - next_permutation
    - lower_bound
    - upper_bound
